Digital Closet

Rust CLI for outfit ideas + Germany-focused shopping links

v1.0.0

Your outfit plan, in one command.

Digital Closet turns your event, season, sizes, and budget into an outfit idea plus clickable store searches (Zalando, H&M, Zara, Jack & Jones, and more).

Works offline for suggestions Links open in your browser No product-page scraping

Smart recommendations

One clean outfit idea from your inputs, designed for fast decision-making.

German retailers

Generates store search links tailored for Germany-focused brands.

CLI-first

Simple commands, predictable output, and a workflow that fits developers.

Quick start

CLI

Run from the cloned repository:

$ cargo run -- recommend wedding winter men --size M --shoe-size 42 --accessories --budget-max 200
View on GitHub Quick Start Guide

Tip: try digital-closet recommend --help for all options.

How it works

1

Install / build

Use Cargo (or your provided binary) to run the tool locally.

2

Enter preferences

Set size, shoes, season, and budget to tailor the recommendation.

3

Open links

Get an outfit idea and safe store searches you can click immediately.
